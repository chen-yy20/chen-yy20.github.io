## Jekyll 简介

Jekyll 是一个简单的静态网站生成器，非常适合创建个人博客。它支持 Markdown 语法，让你可以专注于内容创作。

### 主要特性

- 支持 Markdown 语法
- 自动生成静态网站
- GitHub Pages 原生支持
- 灵活的模板系统

## 如何添加新文章

在 `_posts` 目录下创建新的 Markdown 文件，文件名格式为 `YYYY-MM-DD-title.md`。
